# GTD Quick Filter — Obsidian plugin

Things-style instant tag/context filtering over GTD task notes: hierarchical
tags, AND/OR, status tabs, area/project/date filters, manual ordering with
drag & drop, sub-tasks, sortable views. Works on desktop and mobile.

## Install via BRAT
1. Install the **BRAT** community plugin.
2. BRAT → "Add beta plugin" → `dantetemp/obsidian-gtd-quick-filter`.
3. Enable **GTD Quick Filter** in Community plugins.

BRAT keeps it auto-updated across all your devices.
